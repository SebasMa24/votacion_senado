Carga Masiva de Votantes

La carga de votantes solo se puede realizar de forma masiva mediante un archivo CSV.

El archivo debe contener las siguientes columnas en este orden:

  • Cédula
  • Nombre
  • Apellido
  • Correo
  • Circunscripción
